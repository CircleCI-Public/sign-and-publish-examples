# @circleci/sign-and-publish-example

An example package for demonstrating trusted publishing to [JSR](https://jsr.io/) via GitHub Actions.

## Usage

```ts
import { hello } from "@circleci/sign-and-publish-example";

console.log(hello());
```

## Publishing

This package is published to JSR using trusted publishing through GitHub Actions. See the repository root for CI/CD configuration details.
